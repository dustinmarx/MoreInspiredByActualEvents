Sun Minor Codes

The Java IDL (CORBA) Documentation includes an Exceptions page that describes the differences between CORBA and Java exceptions and explains that "all CORBA system exceptions have a minor code field, a number that provides additional information about the nature of the failure that caused the exception." That page goes onto say that "minor code meanings are not specified by the OMG" and that "each ORB vendor specifies appropriate minor codes for that implementation." One of the most useful aspects of this page is that it lists and describes "some of the most common Sun minor code exceptions ("minor codes thrown by the Java ORB"). I reference these codes and other codes on this page.

These codes typically appear in the form vmcid: SUN minor code: XXX where XXX is the three-digit Sun minor code and "vmcid" stands for Vendor Minor Codeset ID (VMCID). As the table below indicates, the same 3-digit number can appear for entirely different exception conditions. The text associated with these exception messages often provides clues to differentiate between conditions with the same 3-digit number for Sun minor code.
